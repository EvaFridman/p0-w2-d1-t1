Для первой раскладки использовала justify-content:flex-start (прижать к левому краю) и gap (задать расстояние между блоками).

Для второй раскладки использовала justify-content:space-between (задать равные промежутки между строками).

Для третьей раскладки использовала justify-content:center (выравнять блоки по центру по main axis) и  align-items:center (выровнять блоки по центру по cross axis).

Для четвертой раскладки использовала flex-direction:column (поставить блоки в колонку, т.е. вертикально).

Для пятой раскладки использовала align-items:flex-end (прижать к нижнему краю) и min-height (зажать высоту, чтобы было видно, что они прижались к нижнему краю).
