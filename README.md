# p0-w2-d1-t1 · Ряд на флексе

> Тип: короткая отработка · Репозиторий: один отдельный

## Задача

Дан ряд из нескольких блоков. Научиться раскладывать и выравнивать его флексом: менять
направление, прижимать к краям и центру, выравнивать по обеим осям, ставить расстояние через
`gap`. Цель — чтобы пальцы запомнили, что делает каждое свойство контейнера.

```
display: flex  →  элементы в ряд по главной оси

justify-content ──▶ вдоль главной оси (→)
align-items     ──▶ поперёк (↓)

[ A ][ B ][ C ]      gap — расстояние между ними
  flex-direction меняет, где главная ось
```

## Что нужно сделать

Сделать `index.html` с одним контейнером и 4–5 блоками внутри (просто пронумерованные
квадраты — этого достаточно), подключить `style.css`. Дальше — отработать свойства флекс-
контейнера, добиваясь каждой из раскладок ниже. Только CSS, разметку не трогать после того,
как блоки созданы.

Раскладки, которые нужно получить. Делаешь по очереди: добившись очередной, предыдущую не
удаляй, а закомментируй в CSS и пометь комментарием номер релиза (`/* release 2 */`). Так в
финальном PR будут видны все пройденные шаги.

- блоки в строку, прижаты к левому краю, между ними одинаковый `gap`;
- блоки в строку, распределены с равными промежутками между собой (`space-between`);
- блоки по центру контейнера и по горизонтали, и по вертикали;
- блоки в колонку вместо строки;
- один и тот же набор — выровнять по нижнему краю контейнера (чтобы это было видно, контейнеру
  нужно задать высоту больше, чем у блоков, — иначе выравнивать не в чем).

## Критерии приёмки

* Контейнер — `display: flex`, расстояние между блоками задано через `gap`, а не отступами у
  каждого блока.
* Получены все пять раскладок; в `CSS.md` коротко: какое свойство за какую отвечает.
* Понятно (можешь объяснить на словах), где у контейнера главная ось, а где поперечная, и как
  на это влияет `flex-direction`.
